## 验收标准检查清单

### AC1：简化的账号添加流程
- [ ] 添加账号时默认自动生成 SSH 密钥
- [ ] 可以选择生成新密钥、选择已有密钥或暂不配置
- [ ] 自动生成的密钥使用 ed25519 类型
- [ ] 密钥文件名为 `id_ed25519_<账号名>`
- [ ] 密钥注释格式为 `<账号名> <邮箱>`
- [ ] 选择已有密钥时正确列出可用密钥
- [ ] 重复密钥时显示警告但允许继续
- [ ] 添加成功后始终展示公钥内容

### AC2：SSH 密钥管理隐藏
- [ ] 普通模式下主菜单不显示 SSH 密钥管理选项
- [ ] 普通模式下 `git-toolkit ssh` 命令不显示在帮助中
- [ ] 普通模式下 `git-toolkit ssh` 命令仍可调用
- [ ] 列出账号时显示使用的 SSH 密钥信息

### AC3：账号删除流程
- [ ] 删除账号时询问是否删除 SSH 密钥
- [ ] 默认选择不删除密钥
- [ ] 确认删除时同时删除公钥和私钥
- [ ] 删除前验证密钥归属

### AC4：专家模式
- [ ] 通过 `GIT_TOOLKIT_EXPERT_MODE=true` 启用专家模式
- [ ] 专家模式下主菜单显示 SSH 密钥管理选项
- [ ] 专家模式下 `git-toolkit ssh` 命令在帮助中正常显示
- [ ] 专家模式下 SSH 密钥管理功能完全可用

### AC5：配置迁移与兼容性
- [ ] 首次运行时自动检测旧配置
- [ ] 自动迁移旧配置到新格式
- [ ] 迁移前自动备份旧配置
- [ ] 仍支持读取旧格式配置

### AC6：整体质量
- [ ] 代码符合 constitution.md 规范
- [ ] 所有功能有清晰的错误处理
- [ ] 在 macOS 和 Linux 上正常运行
- [ ] 配置文件格式易于手动编辑
- [ ] 操作响应及时，不影响用户体验
